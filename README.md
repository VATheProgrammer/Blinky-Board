# Blinky-Board
This project was made through Hack Club. It will feature a set of multi-colored LEDs that will rapidly blink.  
I made this using EasyEDA, where I designed the PCB Layout and then the actual PCB Design.  
The project features an NE555 timer, which acts as the clock for the amount of blinking that the LEDs do. There are also variable resistors to set the speed of the clock. The clock is then split into 10 separate signals that drive the LEDs.
